Denomination

Quickly calculate an optimal euro denomination breakdown for any amount and see the difference to an existing breakdown.

What this app does

  • Compute the minimal number of notes and coins (in euros) needed to represent an amount.
  • Show the difference between the previous breakdown and the new breakdown.
  • Provide a simple web UI and an API endpoint for programmatic use.

How to use

  1. Enter an amount (in euros) in the denomination page.
  2. Submit to view the breakdown and the difference to the previous breakdown.

Developer notes

The UI is built with Angular. This app supports two calculation modes which can be switched in the UI:

  • Frontend mode — calculation runs in the browser
  • Backend mode — calculation is performed by the backend service at /api/public/denomination/v1 (POST).

Choose the mode using the frontend/backend toggle in the app. The README contains the API request/response JSON shape for backend mode.

See the project README at the repository root for build, run and Docker instructions.
